now

This is a page documenting what I'm currently working on and doing.
This practice is thanks to Derek Sivers' /now page movement.

I’m currently studying a Bachelor of Games and Interactivity and a Bachelor of Animation at Swinburne University of Technology in Melbourne, Australia. It's summer break and other than working, I plan to focus on my own projects to both build up a portfolio and practice skills I feel I’m less strong at.

My goals this break are to:
- continue to grow my website’s content and ecosystem
- create a game using twine and upload it to itch.io
- practice my drawing skills with a focus on anatomy, observational sketching and maybe character design
- build a pc in order to be able to work more efficiently with the tools I need from home
- start illustrating and conceptualising a huge world-building project

Last Updated: 29th October, 2023
